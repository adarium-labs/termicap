# Tutorials

Tutorials are **learning-oriented** guides that take you through a series of steps to complete a project. They focus on learning, not accomplishing a task.

## Available Tutorials

### Beginner

1. **[Getting Started](01-getting-started.md)**
   Detect terminal capabilities in your Ada application.
   Time: ~10 minutes

2. **[Color Detection](02-color-detection.md)**
   Query and use color support levels.
   Time: ~15 minutes

3. **[Cross-Platform](03-cross-platform.md)**
   Handle Unix and Windows terminals.
   Time: ~15 minutes

### Intermediate

4. **[Standards Compliance](04-standards.md)**
   Implement NO_COLOR and FORCE_COLOR support.
   Time: ~15 minutes

5. **[CI/CD Integration](05-ci.md)**
   Detect and handle CI environments.
   Time: ~10 minutes

### Advanced

6. **[SPARK Verification](06-spark.md)**
   Understand the SPARK boundary and proven properties.
   Time: ~20 minutes

## Prerequisites

Before starting, ensure you have:

- Ada compiler (GNAT recommended)
- Alire package manager
- Basic Ada knowledge
